## Development

GlassBox currently provides the buildable project foundation. Product features
are not implemented yet.

Requirements: Node.js 24 and pnpm 11.13.1.

```sh
pnpm install
pnpm dev
```

Run every quality gate with:

```sh
pnpm check
```
